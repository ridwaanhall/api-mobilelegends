OpenMLBB Python SDK

Endpoint: /openmlbb/academy/meta/version

Academy Meta Version Guide

This page explains how to use the OpenMLBB package for the same data as API endpoint /api/academy/meta/version. All examples below use from OpenMLBB import OpenMLBB.

Install

pip install OpenMLBB

Base API

https://mlbb.rone.dev/api

Method

client.academy.meta_version(...)

Query Parameters

Parameter Type Default Description
size int 20 Number of records per page (min 1).
index int 1 Page index (starts at 1).
order str desc Sort order: asc or desc.
lang str en Localization language code.

Python Examples

All examples below return JSON data as Python dictionary.

Example 1: Basic Request

from OpenMLBB import OpenMLBB

client = OpenMLBB()
response = client.academy.meta_version()
print(response)

Example 2: Custom Query

from OpenMLBB import OpenMLBB

client = OpenMLBB()
response = client.academy.meta_version(size=10, index=1, order="asc", lang="en")
print(response)

Example 3: Custom Base URL / Timeout

from OpenMLBB import OpenMLBB

client = OpenMLBB(
    base_url="https://mlbb.rone.dev/api",
    timeout=30,
    user_agent="RoneAI OpenMLBB"
)

response = client.academy.meta_version(lang="en")
print(response)
